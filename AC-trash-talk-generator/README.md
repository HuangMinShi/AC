## Installation and usage
**於終端機輸入指令複製專案**
```git=
git clone https://github.com/HuangMinShi/trash_talk_generator.git
```

**切換至trash_talk_generator**
```=
cd trash_talk_generator
```

**安裝專案套件**
```npm=
npm install -g nodemon
```

**啟動伺服器**
```=
npm run dev
```

**於瀏覽器輸入URL**
```
http://localhost:3000
```
